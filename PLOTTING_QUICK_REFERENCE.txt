===============================================================================
PLOTTING SYSTEM - QUICK REFERENCE
===============================================================================

STANDARD WORKFLOW:
  1. pixi run python runner.py              # Generates outputs/csv/*.csv
  2. python scripts/generate_plots.py --all # Generates figures/{type}/*.png
  3. python scripts/build_pdfs.py --tables  # Compiles Tables_Results PDF

OUTPUT LOCATIONS:
  - figures/mean/              (2 plots: method_comparison, innovation_comparison)
  - figures/parameter/         (2 plots: method_comparison, innovation_comparison)
  - figures/variance/          (3 plots: method_comparison, uncertainty, innovation_comparison)
  - outputs/pdf/               (timestamped PDFs)

GENERATE BY BREAK TYPE:
  python scripts/generate_plots.py --variance
  python scripts/generate_plots.py --mean
  python scripts/generate_plots.py --parameter

GENERATE BY SUBTYPE:
  python scripts/generate_plots.py --single      # Single breaks only
  python scripts/generate_plots.py --recurring   # Recurring breaks only

GENERATE BY TIER:
  python scripts/generate_plots.py --tier1       # Core metrics (RMSE, MAE, Bias, Variance)
  python scripts/generate_plots.py --tier2       # DGP visualizations

COMBINATIONS:
  python scripts/generate_plots.py --variance --tier1
  python scripts/generate_plots.py --mean --single --tier2
  python scripts/generate_plots.py --parameter --recurring

LIST AVAILABLE PLOTS:
  python scripts/generate_plots.py --list

PLOT SPECIFICATIONS:
  - DPI: 300 (publication quality)
  - Font: Sans-serif, 11pt body, 13pt titles
  - Colors: Professional blue (#2E86AB) primary, purple accent, red/teal for coverage
  - Grid: Light gridlines (alpha=0.3) with dashed style
  - All spines removed except bottom/left

INTEGRATION WITH RUNNER:
  pixi run python runner.py --generate-plots  # Run simulations AND generate plots
  pixi run python runner.py --plots-only      # Generate plots from existing results

METHODS COMPARED:
  - Global ARIMA           (Baseline: fit entire sample)
  - Rolling ARIMA          (Adaptive: last 50 observations)
  - GARCH(1,1)             (Volatility targeting)
  - Markov Switching       (Hidden regime detection)
  - Oracle Dummy           (Known break point - upper bound benchmark)

TROUBLESHOOTING:
  - No plots? Check results: ls outputs/csv/aligned_breaks_*.csv
  - Coverage missing? Ensure --break-type variance is specified
  - Innovation comparison missing? Check for multiple innovation types (Gaussian, Student-t variants)
